Welcome to Ypsilon Script

Ypsilon Script (YS) is a simple high-level language that compiles to C++ for microcontrollers. It's designed to make microcontroller development accessible, structured, and type-safe.

Write clean, type-safe code with classes, enums, structs, and pattern matching, then compile to efficient C++ for your microcontroller.

Key Features

  • Strong Static Typing: All variables and functions must be explicitly typed
  • Modern Syntax: Uses fn, mut, self keywords
  • Hardware Types: Built-in hardware abstraction types with automatic setup (100+ types)
  • Unit System: Time (ms, s, us), frequency (Hz), angle (deg), distance (cm, m), speed (rpm)
  • Object-Oriented Programming: Classes with constructors and methods
  • Pattern Matching: Match expressions like Rust for powerful control flow
  • Event-Driven Programming: on start {}, on loop {} blocks
  • No Performance Overhead: Generates efficient C++ code

Quick Example

load <Servo>

// Hardware setup
mut Servo myServo on pin 9

// Main program
on start {
    myServo.attach()
}

on loop {
    myServo.write(90 deg)
    wait 1s
    myServo.write(0 deg)
    wait 1s
}
